現在這個版本可以做什麼？

可以輸入英文句子、播放瀏覽器內建美式發音、錄音、模擬分析，並自動保存最近 5 筆紀錄。

現在的發音分析是真的嗎？

不是。目前是前端模擬分數，目的是確認 UI 和流程。正式版會串接 Azure Speech Pronunciation Assessment。

角色圖片怎麼放？

請放到 assets/characters/，檔名為 character-koala-default.png、character-koala-happy.png、character-koala-angry.png。

為什麼現在先不接 Azure Speech？

因為 Azure 是付費 API。先確認介面、需求、操作流程後，再串接真人語音和真正發音分析會比較省成本。
