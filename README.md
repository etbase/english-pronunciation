# ERIC TING English Pronunciation MVP v2

## 新增內容
- stop.svg（停止錄音）與 analysis.svg（分析發音）已改成白色，適合放在按鈕裡。
- 角色圖片支援：
  - character-koala-default.png（預設）
  - character-koala-happy.png（60 分以上）
  - character-koala-angry.png（60 分以下）
- 「錄音暫存」已改成「下載音檔」。
- 歷史紀錄仍會自動保存最近 5 筆。
- 新增六個分析項目 icon：
  - metric-accuracy.svg（發音準確度）
  - metric-stress.svg（重音）
  - metric-rhythm.svg（節奏）
  - metric-connected-speech.svg（連音）
  - metric-elision.svg（省音）
  - metric-fluency.svg（流暢度）
